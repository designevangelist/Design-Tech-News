ML Engineer – Recommendations

Spotify · Stockholm / Remote

Company

Spotify

Location

Stockholm / Remote

Job Type

Full-time

Salary

$130K–$180K

Spotify’s Recommendations team is hiring an ML Engineer to build and improve the models that power personalized music and podcast discovery for 600M+ users.

Responsibilities

  • Build, train, and deploy machine learning models at scale
  • Collaborate with data scientists and backend engineers
  • Improve recommendation quality through experimentation
  • Monitor model performance and address degradation

Requirements

  • 4+ years of ML engineering experience
  • Proficiency in Python and ML frameworks (PyTorch or TensorFlow)
  • Experience with large-scale data pipelines
  • Strong understanding of recommendation systems or ranking models
